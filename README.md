# Login Form (HTML, CSS & JavaScript)

A simple login form built using HTML, CSS, and JavaScript.

## How it works
- Username can be any value
- Password must be **User123**
- If the password is correct, a welcome message is shown
- If incorrect, an error message is displayed
- Form submission is handled using JavaScript and `preventDefault()`

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project is created for learning basic form handling, DOM manipulation, and event handling in JavaScript.
